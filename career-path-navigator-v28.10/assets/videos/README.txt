Put your video files here (MP4 format recommended).

File naming convention:
  video1.mp4  →  سارة أحمد (or your first interviewee)
  video2.mp4  →  محمد خالد (or your second interviewee)
  video3.mp4  →  مي عصام  (or your third interviewee)

To add more videos:
  1. Drop the .mp4 file in this folder (e.g. video4.mp4)
  2. Copy a <div class="carousel-slide"> block in index.html
  3. Update src="assets/videos/video4.mp4"
  4. Update data-name and data-role attributes
  5. Update the caption-name and caption-role text inside the block
